**Universal x86 Tuning Utility (UXTU)** is a WPF application for controlling hardware on handheld gaming PCs and laptops. The **Hydro UI** is a feature that controls **LCT (Laptop Computer Technology) liquid water coolers** found in certain Tongfang-chassis laptops via **Bluetooth Low Energy (BLE)**.

**User's hardware:** XMG Neo 16 A25 (Uniwill/Tongfang chassis with LCT watercooler)

**Supported watercooler devices:** LCT21001, LCT22002
**Supported brands:** XMG Oasis, PC Specialist, Eluktronics, TUXEDO Aquaris, Schenker
